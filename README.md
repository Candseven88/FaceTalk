# FaceTalk

FaceTalk是一个基于AI的头像动画生成平台，让您能够将静态照片转变为会说话的、生动的头像。

## 主要功能

- **实时肖像生成**：上传肖像照片并生成动画效果
- **语音克隆**：通过短音频样本克隆任何声音并生成语音
- **会说话的头像**：将肖像与音频或文本结合，创建逼真的说话头像

## 技术栈

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- 第三方AI API集成

## 安装与使用

```bash
# 安装依赖
npm install

# 运行开发服务器
npm run dev

# 构建生产版本
npm run build

# 运行生产版本
npm start
```

## 部署

该项目可以轻松部署到Vercel平台。
